# 開明體系 (Enlightened System)

用於股票交易、教學優化與文化推廣的自動化決策支援系統



![Status](https://img.shields.io/badge/status-active-success)




![Python](https://img.shields.io/badge/python-3.11+-blue)



## 📊 項目概述

**開明體系** 是一個全自動化的決策支援系統。

### 🎯 核心功能

1. **股票監控與交易決策** 📈
   - 實時監控台灣股市：聯電 (2303)、慧洋-KY (2637)、和碩 (4938)
   - 技術分析：MA、RSI、MACD 等指標
   - 自動生成買賣信號
   - 每日定時執行（09:15、14:00 台灣時間）

2. **教學流程優化** 🎓
   - 備課時間分析
   - 學生互動評分

3. **客家文化推廣** 🏮
   - 四縣腔文化內容發佈
